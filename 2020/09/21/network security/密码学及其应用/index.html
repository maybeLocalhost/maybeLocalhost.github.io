典型加密标准

优质密码特征：

  • 加密解密工作量由所要求的安全程度决定
  • 密钥集合和加密算法不应过于复杂
  • 执行过程尽量简单
  • 密码中的差错不应具有传播性，也不应影响报文中的其他信息
  • 密文不应比原文大

计算安全的加密方案：

  • 破解密文的代价超出被加密信息的价值
  • 破解密文需要的时间超出信息的有用寿命
