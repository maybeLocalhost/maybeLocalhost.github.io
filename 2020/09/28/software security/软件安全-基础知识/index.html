处理器硬件架构基础

CPU结构

CPU结构包括：寄存器、算术逻辑单元 (ALU)、控制器和内部总线

  • 寄存器提供 CPU 的内部存储，用来暂时存放参与运算的数据及运算结果
  • 算术逻辑单元执行计算机的运算功能
  • 控制器控制计算机各部件工作，包括取指、译码、执行
  • 内部总线将以上部分互连，提供通信机制

保护模式

IA-32 架构 CPU 具有两种工作模式：实模式、保护模式

工作模式受 CR0 控制寄存器的 PE 标志位控制，CPU 刚开始初始化后工作在实模式下，PE 值为 0 ，
