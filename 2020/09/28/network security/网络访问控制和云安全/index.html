鹅鹅鹅

网络访问控制（NAC）

NAC 对登录到网络的用户进行认证，并决定该用户可以访问哪些数据，执行哪些操作，同时可以检查用户的计算机或者移动设备的安全程度。

网络访问控制系统的组成元素

  • 访问请求者 (AR)：尝试访问网络的节点
  • 策略服务器：基于AR的态度和预先定义好的策略，决定授予请求者什么访问权限
  • 网络访问服务器 (NAS)：远程用户系统想连接公司内网时，起到一个访问控制点的作用

网络访问强制措施

  • IEEE 802.1X：一个链接层协议，在一个端口被分配 IP 之前必须强制进行认证
  • 虚拟局域网 (VLAN)：互联的局域网组成
